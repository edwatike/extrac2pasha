Модуль protections
==================

Модуль для обнаружения и обхода различных типов защиты веб-сайтов.

Поддерживаемые типы защит:
- Captcha (reCAPTCHA и другие)
- Cloudflare
- 403 Forbidden
- JavaScript challenge

Методы обхода:
- Selenium WebDriver
- Playwright
- CloudScraper
- Rotating Proxy (через Tor)
- Undetected ChromeDriver
